# 📚 Документация проекта

## 📁 Структура документации

### 🎯 Для администраторов

#### `ADMIN_OUTCOMES_GUIDE.md`

**Полное руководство по созданию исходов в Payload CMS**

Содержит:

- Пошаговые инструкции для всех типов исходов
- Примеры: П1/Х/П2, ТБ/ТМ, ИТБ/ИТМ, ОЗ, Форы, Двойной шанс
- Комбинированные исходы: ОЗ+ТБ, П1+ТБ, 1Х+ТБ
- Частые ошибки и их решения
- Сводная таблица всех типов исходов
- Быстрый старт для новых админов

**Кому:** Администраторам, которые создают исходы через CMS

## 🚀 Быстрый старт

### Для администраторов

1. Откройте `ADMIN_OUTCOMES_GUIDE.md`
2. Следуйте разделу "🚀 Быстрый старт"
3. Создайте первые группы исходов

### Для разработчиков

1. Изучите `OUTCOME_GROUPS_CHEATSHEET.md`
2. Посмотрите примеры JSON-структур
3. Реализуйте логику оценки в `src/lib/prediction-mapping-from-cms.ts`

---
