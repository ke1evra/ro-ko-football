# Анализ полей для П1 (Победа первой команды)

## Все доступные поля в исходе (Outcome)

### ✅ Поля, которые ЕСТЬ в OutcomeGroups.ts:

```
1. name                    - Название исхода
2. comparisonOperator      - Оператор сравнения
3. scope                   - Область применения
4. aggregation             - Способ вычисления
5. range                   - Диапазон (для between)
6. set                     - Множество значений (для in)
7. eventFilter             - Фильтр событий (для exists)
8. outcomeValue            - Значение исхода ← ГЛАВНОЕ ДЛЯ П1!
9. values                  - Значения (для тоталов)
```
